Control your Hue lights from a webpage

How to enable access to your Hue Hub

  1. Visit https://discovery.meethue.com/ to get the IP address of the bridge
  2. Connect to the web page using https://{bridge ip address}/debug}/clip.html
  3. In the URL section type: /api
  4. In the Message Body Section type: {"devicetype":"my_hue_app#{name}"}
  5. Click POST
  6. Go to your Hue bridge and press the button
  7. Click POST again
  8. Copy down the username in the Command Response section
